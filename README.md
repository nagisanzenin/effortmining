# effortmining

> Mine the effort dimension. Spawn every subagent at the effort it deserves — no more, no less.

**Status: research phase.** This repo is being built by a production-grade pipeline run.
The thesis: Claude Code lets you spawn subagents with per-agent reasoning effort
(`low → max`), effort is roughly proportional to token burn, and there is (hypothesis,
under investigation) no built-in calibration deciding how much effort each spawned
agent actually needs. effortmining exists to fill that gap — scientifically, with
A/B-benchmarked numbers, not vibes.

Research docs land in `docs/research/`. Benchmark methodology, harness, and measured
results follow after Gate A.
